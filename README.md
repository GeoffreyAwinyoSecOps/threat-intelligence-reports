# Threat Intelligence Reports

This repository contains Cyber Threat Intelligence (CTI) reports and detection engineering work focused on analyzing adversary behavior and translating intelligence into actionable detections.

## Objectives
- Analyze adversary tactics, techniques, and procedures (TTPs)
- Map behaviors to MITRE ATT&CK
- Develop detection logic for SIEM environments
- Provide actionable guidance for SOC teams

## Reports

### ISO MOTW Bypass (Qakbot Tradecraft)
- Technique: Mark-of-the-Web Bypass via ISO delivery
- MITRE ATT&CK: T1553.005, T1027, T1204.002
- Includes detection logic and SOC guidance

## Detection Content
Detection rules and logic derived from intelligence analysis are included in the `/detections` folder.
